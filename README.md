This is a repo for a QR generation app. The idea is to have a free website to get customized QR codes that can be downloaded and used freely.

I started this project because of interest in QR codes in general, but now I am taking this chance to have some experience with deploying actual apps (even if it's on a small scale)

## Features in progress:
- Have QR codes of any color even with transparent color (with graphic designers in mind : ) )
- Having an option to add photo onto or interleaved withing the QR code
